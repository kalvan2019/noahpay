## A

|                     |                                                           |
| ------------------- | --------------------------------------------------------- |
| accountName         | 账户名称                                                  |
| accountPersonIdCode | 账户人（持卡人）身份证号码                                |
| accountType         | 账户类型，一般分为对公和对私 （目前还有 bankAccountType） |
| activeWxRate        | 是否开通 0 费率通道 'ZERO':开通 'NORMAL':不开通           |
| address             | 门店地址                                                  |
| aliRate             | 支付宝费率                                                |
| appId               | 接入方 ID，渠道 ID，每个接入开放平台的一个渠道发放一个    |
| authCode            | 用户付款码                                                |
| auditStatus         | 审核状态                                                  |
| auditStatusName     | 审核状态名称                                              |
